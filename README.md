# TaskQRcode

This is my **task project**: a simple Certificate Verification System built with **Kotlin + Jetpack Compose + Firebase + ZXing**.

## Features
- Enter student name, program, date
- Save certificate to **Firebase Firestore**
- Generate a **QR code** for each certificate
- Scan QR to verify certificate

## Tech
Kotlin, Jetpack Compose, Firebase Firestore, ZXing, Navigation Compose
